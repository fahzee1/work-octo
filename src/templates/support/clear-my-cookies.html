How do our Cookies work?

We hold 3 tracking specific cookies. Agent Cookie, Source Cookie, and Affiliate Key Cookie.

  • Agent Cookie - The agent cookie stores our agent code generally resembling a00000. This tells us who should get the credit for a lead that is generated thru ProtectAmerica.com
    Accessed by appending agent=a00000 to the URL string.
  • Source Cookie - The source cookie is simply the name associated to the Agent Cookie.
    Automatically assigned via a database called.
  • Affiliate Key Cookie - The affiliate key cookie or affkey is a cookie to allow affiliates the ability to do one more level of tracking.
    Accessed by appending affkey=XXXXXX to the URL string.
